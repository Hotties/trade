(선택) N일 평균 거래량,>,M일 평균 거래량

1. 주식 가격 데이터 가져오기
2. 당일의 5일평균, 20일평균, 60일평균, 120일 평균을 구함
2-1. 만약 5일>20일>60일>120일 의 경우 그래프가 우상향이므로 선정
2-1-1. 거래량을 측정하여 많을 경우 선택 
(조건 구분,구체적인 기준,설명
단기 모멘텀,당일 거래량≥5일 평균 거래량×150%,최근 일주일 평균보다 1.5배 이상 거래량이 증가했다는 것은 단기적으로 매수세가 강하게 유입되었다는 증거입니다.
장기적 활발도,5일 평균 거래량≥20일 평균 거래량,최근 단기적인 거래량 평균이 중기 평균보다 많다는 것은 거래의 활발함이 증가 추세에 있음을 나타냅니다.)

2-1-2. 아닐경우 (추천 전략,구체적인 매수 조건,설명
추세 눌림목 매수,종가≤20일 이평선 ($\mathbf{\text{20일 이평선}}$에 근접하거나 닿을 때),"정배열 상태에서 주가가 일시적으로 **20일 이평선(생명선)**까지 하락 조정받을 때, 이평선이 강력한 지지선 역할을 할 것으로 기대하고 매수 진입 시점을 포착합니다.")

2-2. 아닐경우 선택사항에서 제외

3. 선택된 주식을 증권사 API를 통해 접속
4. 금액 투자 후 가격이 일정 퍼센트 올라가면 매매, 떨어지면 매도
(기준,구체적인 기준,설명
목표 수익률,3%∼5% 상승 시 매도,투자 성향에 따라 단기 목표 수익률을 설정합니다. 주가가 지정된 비율만큼 상승하면 자동으로 매도하여 수익을 확보합니다.
이동평균선 이탈,5일 이평선을 하향 이탈 시 매도,"주가가 5일선 아래로 내려갈 경우, 단기적인 상승 모멘텀이 꺾였다고 판단하여 익절 후 관망합니다.")

(기준,구체적인 기준,설명
손실 제한 비율,3%∼5% 하락 시 매도,원금 손실을 최소화하기 위해 반드시 설정해야 하는 기준입니다.
핵심 지지선 이탈,20일 이평선을 하향 이탈 시 매도,매수한 주가가 20일 이평선 아래로 떨어지면 중기적인 추세 이탈로 간주하고 손절하여 추가 손실을 방지합니다. (가장 중요한 손절 기준))
